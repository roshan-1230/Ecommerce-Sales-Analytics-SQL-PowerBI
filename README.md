# 🛒 E-Commerce Sales Analytics Project

An end-to-end Data Analytics project built using **MySQL** and **Power BI** to analyze 50,000 e-commerce transactions and generate business insights.

## 🚀 Project Overview

This project focuses on:
- Database Design & Normalization
- Advanced SQL Analysis
- Business KPI Reporting
- Interactive Power BI Dashboard

## 🛠️ Tools Used

- MySQL
- SQL
- Power BI
- GitHub

## 📊 SQL Concepts Used

- Joins
- Aggregate Functions
- Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
- CTEs
- Views
- Stored Procedures
- Data Normalization

## 📈 Dashboard Features

- Total Revenue
- Total Transactions
- Average Order Value
- Total Customers
- Revenue by Category
- Revenue by Country
- Monthly Revenue Trends
- Payment Method Analysis
- Top Customers Analysis

## 🗄️ Database Structure

- Customers
- Categories
- Transactions
